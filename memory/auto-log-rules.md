# 自动记录规则

## 触发条件

### 1. 对话结束时自动写
当对话中出现以下关键词时，自动记录：
- **任务开始**："开始..."、"要做..."、"任务..."、"需求..."
- **任务完成**："完成了"、"done"、"搞定"、"结束"
- **时间记录**：自动附加当前时间

### 2. 定时汇总 (每天 22:00)
- 整理当天所有任务
- 生成时间线摘要
- 记录未完成任务

## 记录格式
```markdown
## 任务记录

### 任务名
- 开始时间: HH:MM
- 完成时间: HH:MM (如已完成)
- 状态: 进行中/已完成
- 备注: (可选)
```

## 文件位置
- 每日记录: `memory/YYYY-MM-DD.md`
- 长期记忆: `MEMORY.md` (重要事项)
